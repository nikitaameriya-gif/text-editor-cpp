# C++ Text Editor Project

## Description
A simple console-based text editor built in C++ using file handling and OOP concepts.

## Features
- Create a file
- Write to file
- Append content
- Read file
- Menu-driven interface

## Concepts Used
- File Handling (fstream)
- Object-Oriented Programming
- Menu-driven logic
- String handling

## How to Run
```bash
g++ main.cpp TextEditor.cpp -o texteditor
./texteditor
